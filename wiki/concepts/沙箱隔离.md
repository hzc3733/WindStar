## 定义

沙箱隔离是一种安全机制，通过将Agent运行在隔离的容器环境中，防止恶意操作或资源滥用影响系统其他部分。

## 配置选项

| 模式 | 说明 |
| --- | --- |
| `off` | 无沙箱，Agent直接运行在主机环境 |
| `all` | 始终沙箱隔离，每个Agent独立容器 |

## 高级配置

```json
{
  "sandbox": {
    "mode": "all",
    "scope": "agent",
    "docker": {
      "setupCommand": "apt-get update && apt-get install -y git curl"
    }
  }
}
```

## 使用场景

- 高风险操作隔离
- 多租户环境数据隔离
- 资源受限执行环境
