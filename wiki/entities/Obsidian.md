## 定义

Obsidian 是一款本地优先的 Markdown 双链笔记工具，核心特性是通过 `[[双链]]` 语法实现知识节点间的互联，并通过图谱可视化展示知识网络结构。

## 关键信息

- **定位**：本地知识库核心工具，支持双链笔记与图谱可视化
- **插件生态**：支持 Local Rest API 插件，为外部工具提供读写笔记库的接口
- **工作流**：配合 Web Clipper 插件可抓取网页/视频内容，转化为结构化 Markdown 笔记
- **与 LLM-Wiki 的关系**：作为 Karpathy LLM-Wiki 理念的核心承载工具
