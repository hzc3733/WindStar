## 核心摘要

本视频介绍了如何将 Bilibili 视频转化为本地 AI 可理解的知识库。核心工具链包含四个组件：Obsidian（本地知识库核心）、Local REST API 插件（提供外部读写接口）、karpathy-llm-wiki-vault 模板（标准化知识管理结构）、以及 Bilibili Web Clipper 浏览器扩展（抓取 B 站视频原数据与字幕）。工作流程为：B 站视频 → Web Clipper 抓取 → Markdown 存入 vault → LLM Wiki 自动完成双向链接与知识图谱构建。
