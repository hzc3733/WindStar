## 核心摘要

本视频介绍了如何将B站视频转换为本地 LLM-Wiki 知识库的方法。核心展示了 Karpathy 提出的 LLM-Wiki 理念——利用 Obsidian 作为本地知识库，通过 Web Clipper 抓取视频并转化为 Markdown 格式，实现知识互联。由于 Obsidian Web Clipper 无法解析B站页面结构，B站up主小陈师傅开发了专用的 Bilibili Web Clipper 浏览器扩展，可一键将B站视频转化为结构化笔记，配合 Karpathy-LLM-Wiki-Vault 模板自动完成双向链接与知识图谱构建。
