## 核心摘要
本视频由UP主"MIP耀"介绍如何将B站视频整理至本地LLM-Wiki知识库。核心工具链为：Obsidian作为本地知识库核心 + Local REST API插件提供外部接口 + Karpathy-LLM-Wiki-Vault模板提供标准化结构 + Bilibili Web Clipper浏览器扩展打通最后一环。该方案解决了B站视频无法使用Obsidian Web Clipper直接抓取的困境，实现了从视频收藏到AI可理解知识库的完整闭环。
